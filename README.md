# AI Google Drive Agent

Conversational AI agent that intelligently searches Google Drive files using natural language.

## Features

- AI-powered file discovery
- Natural language understanding
- Google Drive API integration
- FastAPI backend
- Streamlit frontend
- LangChain + Groq LLM

## Tech Stack

- Python
- FastAPI
- Streamlit
- LangChain
- Groq
- Google Drive API
